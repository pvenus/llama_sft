# 요구 사항
- 최종 출력 json : {"name":{function}, "datas":["<s><o><v>"], {"tags":[{"<s>": [], "<o>": [], "<v>":[]}]
- 리스트에서 사용하는 표현 방식 추출
- 필수가 아닌 요소 지금/바로 등 기능 호출에 불필요한 표현 제외
- 온 하도록 도와줄거지 -> 온 과 같이 중요 단어로 추출
- 에이아이 모드, 에이아이 기능 이런건 <s1><s2> 로 분리하고 <s1> : [에이아이], <s2>: [모드,기능]
- datas의 동일 한 데이터는 하나로 표현
- datas에 사용 되지 않는 tag는 tags 데이터에 포함시키지 않음
